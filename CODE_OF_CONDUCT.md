# Code of Conduct

## Our Pledge
We pledge to make participation in this project a harassment-free experience
for everyone.

## Our Standards
Examples of behavior that contributes to a positive environment include:
- Using welcoming and inclusive language
- Being respectful of differing viewpoints

## Enforcement Responsibilities
Project maintainers are responsible for clarifying and enforcing standards.

## Scope
This Code of Conduct applies within project spaces and public spaces.

## Enforcement
Instances of abusive behavior may be reported to the project team.

## Enforcement Guidelines
Maintainers may remove, edit, or reject contributions that violate standards.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
